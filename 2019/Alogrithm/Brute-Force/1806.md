#  부분합

> **문제분류:Brute force**
>
> **문제번호:1806**

> solution 1.
>
> 문제의 크기:  N (10 ≤ N < 100,000)과 S (0 < S ≤ 100,000,000)
>
> 시간제한: 1초
>
> 
>
> 이 문제는 수들의 합(2)문제에서 합이 s이상 되는 부분집합의 합에서 가장 작은 길이의 부분집합의 길이를 출력하는 문제 입니다.
>
> 기존의 수들의 합(2) 코드에서 부분집합의 합이>=s 이상인 경우에 right-left+1을 해주어 length를 구해주고 비교해가면 가장 짧의 길이의 부분합을 구할 수 있습니다.
>
