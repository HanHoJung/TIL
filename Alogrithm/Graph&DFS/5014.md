# 스타트링크

> **문제분류:그래프와 bfs**
>
> **문제번호:5014**

> solution 1.
> 문제 조건:   F, S, G, U, D가 주어진다. (1 ≤ S, G ≤ F ≤ 1000000, 0 ≤ U, D ≤ 1000000) 
> 제한시간:1초
>
> 이 문제는 강호가 S층에서 G층으로 가기 위해 눌러야 하는 버튼의 수의 최솟값을 출력한다.
>
> 문제의 조건은 다음과 같습니다.
>
> -  총 F층
> - G:도착 위치
> - S:강호의 현재 위치
> - U:위로 U층 이동
> - D:아래로 D층 이동
>
> 
>
> 우선 이 문제를 bfs를 풀 수 있는 이유는 다음과 같습니다.
>
> - 정점의 정의
>
>   엘리베이터의 층
>
> - 간선의 정의
>
> ​      층과 층의 연결
>
> - U와 D연산 가중치 모두 1
>
> 
>
> BFS를 돌려주면 간단하게 풀 수 있는 문제 입니다.
>
