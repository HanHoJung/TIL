# 1,2,3 더하기

> **문제분류:다이나믹 프로그래밍**
>
> **문제번호:9095**

> solution 1.
>
> 문제의 크기:  N (1 ≤ N ≤ 11) 
>
> 제한시간:1초
>
>
>
> 점화식의 정의: 정수 n을 1, 2, 3의 합으로 나타내는 방법의 수
>
> 0+0+...+0 = n
>
> 마지막 자리에 올 수 있는 수는 1,2,3
>
> 마지막에 1이 오는 경우
>
> 마지막 자리 제외한 합:n-1 + 마지막 자리:1 = n
>
> 마지막 자리 제외한 합:n-2 + 마지막 자리:2 =n
>
> 마지막 자리 제외한 합:n-3+ 마지막 자리: 3 = n
>
>
>
> D[N] = D[N-1]+D[N-2]+D[N-3]
