# 가장 긴 감소하는 부분 수열

> **문제분류:다이나믹 프로그래밍**
>
> **문제번호:11722**

> solution 1.
>
> 문제의 크기:수열의 크기 N (1 ≤ N ≤ 1,000),  수열 A를 이루고 있는 Ai (1 ≤ Ai ≤ 1,000)
>
> 제한시간:1초
>
> 이 문제는 수열이 주어졌을때 부분 수열의 최대길이를 구하는 문제 입니다.
>
> 가장 긴 증가하는 부분 수열에서 다음과 같은 조건만 바꾸어 주면 됩니다.
>
> ```
> (0<=j<i<n)
> dp[i]
> 
> if(arr[j]>arr[i] && dp[i]<dp[j]+1)
> 	dp[i] = dp[j]+1
> 
> (0<=j<i<n)
> dp[i] = max(dp[j]+1)
> ```
>
