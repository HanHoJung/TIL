# 로또

> **문제분류:Brute force**
>
> **문제번호:6603**

> solution 1.
>
> 문제의 크기:  k (6 < k < 13)
>
> 시간제한: 1초
>
>
>
> 이 문제는 k 개의 자리에서 6자리를 뽑는 수열의 수를 구하는 문제입니다.
>
> __  __ __ __ __ __ (k자리)
>
> 1   1    1   1    1   1 (수에 포함) 
>
> 0   0    0   0    0   0(수에 포함 되지 않음)
>
>
>
> 1. 모든 경우의 수 구해보기
>
>    13C6 가지
>
> 2. 가능한 모든 방법 해보기
>
>    ```
>    k=7
>    0111111
>    1011111
>    1101111
>    1110111
>    1111011
>    1111101
>    1111110
>    
>    ```
>
> 3. 각각의 방법
>
>    순열
>
>
>
> 시간복잡도 O(kC6 * n)
>
